## Tier 1 (20 equations — 100% required)
Files: tier1_01.tex through tier1_20.tex
Coverage: Greek letters, sub/superscripts, fractions, integrals,
operators, font commands, accents, delimiters.

## Tier 2 (30 equations — ≥95% required)
Files: tier2_01.tex through tier2_30.tex
Coverage: environments (align, cases, matrix), overline/underline,
binom, text in math, operator names, arrows.

## Tier 3 (10 equations — best-effort)
Files: tier3_01.tex through tier3_10.tex
Coverage: complex expressions, mhchem (unsupported), xrightarrow.

## What shipped

Phase 9.5.1 installed the cross-viewer vector consistency CI
infrastructure: text-as-path export, 12 deterministic source
fixtures, 3 render runners, a 3-tier comparison metric pipeline,
and a GitHub Actions workflow.

### T1 — Text-as-path
- PlotRenderer gains `setTextAsPath(bool)` mode
- All `drawText` calls routed through `drawTextOrPath` helper
- SVG/PDF export automatically enables text-as-path
- Screen rendering unchanged (textAsPath=false)

### T2 — 12 source fixtures
- Deterministic .lumen.json files in tests/export/fixtures/source/
- Generated by tests/export/generate_fixtures.py (reproducible)
- Covers: line, log, scatter, bar, heatmap, contour, errorbar,
  violin, 3D surface/scatter fallbacks

### T3-T5 — Render runners
- playwright_runner.py: headless Chromium via Playwright 1.49.0
- pdftocairo_runner.py: PDF→PNG via poppler-utils
- inkscape_runner.py: SVG→PNG via Inkscape CLI

### T6 — compare.py
- 3-tier metric gate (ADR-056): MS-SSIM, PSNR, CIEDE2000
- JSON report output with gate pass/fail decision
- Dependencies: scikit-image, numpy, colour-science, Pillow

### T7 — test_vector_consistency.py
- pytest parameterized: 12 fixtures × 3 viewers = 36 test cases
- Fixture validation: existence, valid JSON, schema check
- Compare module import verification

### T8 — vector-consistency.yml
- Ubuntu 22.04 runner with Playwright 1.49.0 pinned
- Diff artifact upload on failure
- macOS smoke test placeholder
